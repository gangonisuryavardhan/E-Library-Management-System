# 📚 E-Library Management System

## Overview

The E-Library Management System is a web-based application developed using ASP.NET, C#, SQL Server and ADO.NET.

This application helps librarians manage books, members and book transactions efficiently.

---

# Features

✅ User Registration

✅ User Login

✅ Admin Login

✅ Book Management

✅ Author Management

✅ Publisher Management

✅ Issue Books

✅ Return Books

✅ Member Management

✅ Search Books

✅ Dashboard
